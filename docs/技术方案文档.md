# ANIDOG 番剧自动下载管理系统 - 技术方案文档

## 1. 项目概述

### 1.1 项目简介
ANIDOG（AniDog）是一个基于Web的番剧自动下载管理系统，通过RSS订阅和智能规则匹配，实现番剧的自动追踪、下载和整理功能。

### 1.2 技术目标
- 构建高性能、可扩展的番剧管理系统
- 实现实时通知和状态更新
- 支持多种下载器集成
- 提供友好的用户界面和交互体验

### 1.3 系统特点
- **前后端分离架构**：提高开发效率和系统可维护性
- **实时通信**：WebSocket支持实时状态推送
- **模块化设计**：功能解耦，易于扩展
- **容器化部署**：支持Docker一键部署

#### 3.2.2 核心流程
```python
async def check_rss_updates():
    """检查RSS更新的核心流程"""
    # 1. 获取所有启用的RSS源
    rss_feeds = get_enabled_rss_feeds()
    
    # 2. 并发检查更新
    tasks = [parse_rss_feed(feed) for feed in rss_feeds]
    results = await asyncio.gather(*tasks)
    
    # 3. 应用过滤规则
    for items in results:
        filtered_items = apply_filters(items, feed.filter_rules)
        
    # 4. 触发下载任务
    for item in filtered_items:
        create_download_task(item)
```

class RenameEngine:
    def __init__(self, pattern="{anime_title} - S{season:02d}E{episode:02d}"):
        self.pattern = pattern
    
    def rename_file(self, file_info):
        """根据模板重命名文件"""
        return self.pattern.format(
            anime_title=file_info.anime_title,
            season=file_info.season,
            episode=file_info.episode
        )
```

class NotificationService:
    def __init__(self):
        self.channels = []
    
    def register_channel(self, channel):
        self.channels.append(channel)
    
    async def send_notification(self, event, data):
        tasks = [ch.send(event, data) for ch in self.channels]
        await asyncio.gather(*tasks)
```
